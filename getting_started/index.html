...
< Location: /api/v1/result/c94480a6-9e03-4ca1-b458-b04ea7603551
...

Please note the Location URI for further pollling.

Poll the result

curl -vvvv http://localhost:8080/api/v1/result/c94480a6-9e03-4ca1-b458-b04ea7603551
...
< HTTP/1.1 202 Accepted
...

HTTP Code 202 indicates that the query is running and you should retry in few seconds.
